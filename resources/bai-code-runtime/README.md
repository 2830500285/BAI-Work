# BAI Code Runtime

Bundled BAI Code runtime for BAI Work macOS Intel builds.

- BAI Code: 0.9.1
- Python: CPython 3.11.15 x86_64
- Source installer: https://raw.githubusercontent.com/BAI-labs/BAI-tools/refs/heads/main/scripts/baicode_install.sh

Use `bin/baicode` so the bundled Python and site-packages are selected with relative paths.

# BAI Android

Minimal Android WebView wrapper for `https://chat.b.ai/chat`.

- App name: `BAI`
- Package name: `ai.b.bai`
- Icon source: `../src/asset/img/bai-work.png`
- Build target: Android SDK 35

GitHub Actions builds the installable debug APK and uploads it as the `BAI-android-debug-apk` artifact.

\chapter*{版本说明}
本白皮书是 BAI Work 项目的第一版完整技术白皮书。内容基于当前仓库实现、项目 README、构建配置、运行时适配代码、资源目录说明、EBAI 映射服务以及公开 BAI Code 安装资料整理。文档采用保守口径: 能从代码和资源核对的内容写为当前事实, 官方没有公开的 BAI Code 桌面本地 HTTP service/session/event/permission/question 协议写为待确认边界。

本文不包含真实 API key、token 或任何凭据。涉及环境变量的示例均使用占位符, 例如 BAI\_API\_KEY=your-key。

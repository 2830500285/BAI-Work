# BAI Work 0.1.1

This release rebuilds all desktop targets from one source tag. Native artifacts
are available for macOS Intel x64, macOS Apple Silicon arm64, and Windows x64.

## Downloads

- macOS Intel: `BAI-Work-0.1.1-mac-x64.dmg` or `.zip`
- macOS Apple Silicon: `BAI-Work-0.1.1-mac-arm64.dmg` or `.zip`
- Windows x64: `BAI-Work-0.1.1-win-x64.exe`

## Runtime

- Mac Intel includes a self-contained BAI Code 0.9.1 runtime.
- Mac Apple Silicon and Windows include platform-specific official BAI Code
  wheelhouses for Python 3.10-3.13.
- On first use, Apple Silicon and Windows create a user-local virtual
  environment and install BAI Code offline from the packaged wheelhouse.
- Windows wheelhouses include the Windows-only `colorama` dependency required
  by `tqdm`.

## Included

- Live task progress, tool-step summaries, cleaned final responses, and
  estimated token usage.
- BAI provider configuration without bundled credentials.
- EBAI commands, skills, agents, common rules, and disabled-by-default hooks.
- Native package validation and packaged-runtime health smokes on GitHub-hosted
  Intel Mac, Apple Silicon, and Windows runners.

## Signing

The macOS artifacts are ad-hoc signed and are not notarized. The Windows NSIS
installer is unsigned. Public trusted signing still requires Apple Developer ID
and Windows code-signing credentials.

Checksums are published beside the installers as `SHA256SUMS-*.txt` assets.
